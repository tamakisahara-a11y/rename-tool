撮影時の連番（_01など）はツールが無視するため、自由につけて構いません。

  • ファイル名にスペースを入れないでください
  • 対応拡張子：jpg / jpeg / png / tif / tiff

ファイル名は 品番_種別_連番.拡張子 の形式にしてください。

画像の内容 ファイル名の例

グループ全種の集合画像 MOB0001_全員集合_01.jpg（代表品番1つ）
一部のセット写真 MOB0001_MOB0002_MOB0003_セット集合_01.jpg（対象品番を全て列挙）

連番の割り当て順

ツールは以下の順でJANコードの連番を割り当てます。撮影時の連番（_01など）はツールが無視するため、自由につけて構いません。

  1. 全員集合
  2. セット集合
  3. 個別

注意事項

  • 品番は在庫管理表の「品番」列と完全一致させてください
  • 全員集合・セット集合は、ツールが自動で各JANコード分に複製します
  • ファイル名にスペースを入れないでください
  • 対応拡張子：jpg / jpeg / png / tif / tiff
